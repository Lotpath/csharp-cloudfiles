Csharp-cloudfiles

C#/.NET API for Rackspace Cloud Files

Download this project as a .zip file Download this project as a tar.gz file

Description

This is a .NET/C# interface into the Rackspace CloudFiles service. Cloud Files is a reliable, scalable and affordable web-based storage hosting for backing up and archiving all your static content. Cloud Files is the first and only cloud service that leverages a tier one CDN provider to create such an easy and complete storage-to-delivery solution for media content.
